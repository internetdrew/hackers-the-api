Hackers, The Unofficial Movie API

Welcome to Hackers, The Unofficial Movie API

Based on the 1995 movie Hackers, including characters, organizations, and quotes.

API Endpoints

Characters

  • GET /api/v1/characters: Returns a list of characters.
  • GET /api/v1/characters/{id}: Returns specific character details.
  • GET /api/v1/characters/:id/quotes: Returns character quotes.

Organizations

  • GET /api/v1/organizations: Returns a list of prominant organizations in the movie.
  • GET /api/v1/organizations/{id}: Returns details about a specific organization.

Hacks

  • GET /api/v1/hacks: Returns a list of hacks executed in the movie.
  • GET /api/v1/organizations/{id}: Returns details about a specific organization.

Quotes

  • GET /api/v1/quotes: Returns a list of famous quotes.
  • GET /api/v1/quotes/{id}: Returns details about a specific quote.
