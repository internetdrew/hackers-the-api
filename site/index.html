API Endpoints

Characters

  • GET /api/v1/characters returns a list of characters.
  • GET /api/v1/characters/:id returns information on a particular character.
  • GET api/v1/characters/:id/quotes returns famous lines from particular characters.
  • GET api/v1/characters/:id/hacks returns hacks a character contributed to and was targeted by.

Organizations

  • GET /api/v1/oranizations returns a list of organizations.
  • GET /api/v1/oranizations/:id returns information on a specific organization.

Hacks

  • GET /api/v1/hacks returns a list of hacks.
  • GET /api/v1/hacks/:id returns information on a specific hack.

Quotes

  • GET /api/v1/quotes Returns a list of famous quotes.
  • GET /api/v1/quotes/:id Returns a specific quote.
